# Karthik S - Portfolio

Personal portfolio of Karthik S, built with React and Tailwind CSS.

## Overview
Karthik’s Portfolio – A sleek, single‑page React application showcasing personal information, skills, projects, education, and contact details. Built with Tailwind CSS, Lucide‑React icons, and a focus on premium visual aesthetics (dark/light mode, glass‑morphism, animated gradients). Fully responsive, SEO‑friendly, and ready for deployment.
